Este projeto utiliza inteligência artificial para otimizar a receita de Misto Quente, ajustando ingredientes e processos culinários com base nas preferências alimentares dos usuários.

## Funcionalidades:
1. Escolher diferentes variações de Misto Quente (vegetariano, vegano, clássico).
2. Exibir receitas otimizadas com base nas preferências dos usuários.

### Requisitos do Projeto:
- PHP 7 ou superior
- Navegador compatível com HTML5 e CSS3

### Passos para execução:
1. Configure um servidor PHP local.
2. Coloque o arquivo `misto_quente.php` na raiz do servidor.
3. Acesse o arquivo pelo navegador com o parâmetro `tipo` (ex: `?tipo=vegano`).
4. O estilo pode ser ajustado no arquivo `style.css`.
